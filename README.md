# 従業員管理システム（Employee Management System）

## 概要
本プロジェクトは、Java（Swing）を用いて作成した **デスクトップ型の従業員管理システム** です。  
ログイン機能、新規ユーザー登録、従業員情報の追加・修正・削除・検索といった基本的な管理機能を備えています。

Java の基礎文法、オブジェクト指向、GUI プログラミング、および Maven を用いた依存関係管理の理解を目的として開発しました。

---

## 主な機能

### 1. ログイン機能
- ユーザー名・パスワードによる認証
- 認証失敗時のエラーメッセージ表示
- サンプルユーザーによる即時ログイン可能

### 2. 新規ユーザー登録
- ユーザー名の重複チェック
- パスワード確認入力
- 登録後、ログイン画面へ戻る仕様

### 3. 従業員管理機能
- 従業員一覧表示（JTable）
- 新規従業員の追加
- 従業員情報の修正
- 従業員情報の削除
- あいまい検索（全項目対象）

### 4. 右クリックメニュー対応
- JTable 上での右クリックによる「修正」「削除」操作

---

## 使用技術

- **言語**：Java 17
- **GUI**：Java Swing
- **ビルドツール**：Maven
- **ライブラリ**：
  - Lombok（@Data / @AllArgsConstructor / @NoArgsConstructor）
- **IDE**：IntelliJ IDEA

## プロジェクト構成
EmployeeSystem
├─ src
│  └─ main
│     └─ java
│        └─ com.employee_sys
│           ├─ App.java               // アプリ起動クラス
│           ├─ ui
│           │  ├─ LoginUI.java        // ログイン画面
│           │  ├─ RegisterUI.java     // 新規登録画面

│           │  └─ AddEmployeeUI.java  // 従業員追加・修正画面

│              ├─ User.java           // ユーザー情報Bean
│              └─ Employee.java       // 従業員情報Bean
└─ pom.xml

---

## 実行方法

1. JDK 17 をインストール
2. 本プロジェクトをクローン
3. IntelliJ IDEA で Maven プロジェクトとして開く
4. `App.java` を実行
